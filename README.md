# Starship Finder

## Installation

Navigate inside project folder and run these commands:

```bash
cp .env.example .env
```
```bash
composer install
```

```bash
npm install
```

```bash
npm run dev
```

```bash
php artisan key:generate
```

```bash
php artisan serve
```

Then go to the application link given by the last command. (http://127.0.0.1:8000)
